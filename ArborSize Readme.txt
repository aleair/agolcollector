# Data Collector for ArborSize

## Description
Data Collector for Arborsize is a web application designed to collect spatial data and store it in ArcGIS Online (AGOL) using ArcGIS API for JavaScript. It allows users to add records with attributes such as diameter to a feature layer hosted on AGOL. The application provides a map interface with satellite basemap, a search functionality, and an editor widget for feature editing.

## Features
- Satellite basemap for better visualization of spatial data.
- Locate button to quickly find the user's current location on the map.
- Search widget to search for specific locations or addresses.
- Editor widget for adding new features with custom attributes (e.g., diameter).

## Usage
1. Open the web application in a web browser.
2. Click the "Add Record" button to open a new webpage for adding records to the feature layer.
3. Fill out the diameter value in the form and submit to add a new feature to the map.
4. Use the locate button to find your current location on the map.
5. Use the search widget to search for specific locations or addresses.

## Installation
No installation is required to use the web application. Simply open the provided HTML file in a web browser to access the application.

## Dependencies
- ArcGIS API for JavaScript v4.26
- ArcGIS Online feature layer service
